# Clinical Teaching System

بوابة إلكترونية لكلية الطب البشري تهدف إلى تنظيم توزيع الطلاب على المستشفيات والأقسام والأطباء بشكل سلس.  
يوفر النظام برامج دوام للطلاب والأطباء، ويتيح لكل طبيب رؤية مجموعته الخاصة من الطلاب.  
كما يدعم تسجيل الحضور والغياب، بالإضافة إلى تقييم الطلاب للأطباء وتقييم الأطباء للطلاب.

## ✨ المزايا الرئيسية
- تقسيم الطلاب إلى مجموعات رئيسية وفرعية.
- عرض جداول الدوام للطلاب والأطباء.
- تسجيل الحضور والغياب بشكل رقمي.
- نظام تقييم متبادل بين الطلاب والأطباء.
- تكامل مع Firebase و JWT Authentication.

## 🛠️ التقنيات المستخدمة
- ASP.NET Core
- Firebase
- JWT Authentication
- SQL Server

1. استنسخ المشروع:

   git clone https://github.com/SalahNemer/ClinicalTeachingSystem-P.git
2-ادخل إلى المجلد:
cd ClinicalTeachingSystem-P

dotnet run
